Customer Churn Intelligence System

Overview : 

This project builds an AI-powered Customer Churn Intelligence System to help telecom businesses predict customer churn and take proactive retention actions.
The system analyzes customer data such as demographics, service usage, billing, and support interactions to identify customers at risk of leaving.



Objectives : 

Predict customer churn using machine learning
Assign churn probability to each customer
Segment customers into Low, Medium, and High risk
Identify key factors driving churn
Provide actionable business insights and retention strategies

Tech Stack : 

Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn


Models Used : 

Logistic Regression
Decision Tree
Random Forest (Final Model)

Evaluation Metrics: 

Confusion Matrix
F1-Score (primary metric due to class imbalance)
Cross-validation (5-fold)




Business Insights : 


Customers with low tenure are more likely to churn
High monthly charges increase churn probability
Month-to-month contracts have the highest churn rate
Lack of technical support leads to customer dissatisfaction


🎯 Retention Strategies

🔴 High-Risk Customers
Offer discounts and retention deals
Provide priority customer support
Encourage long-term contracts

🟡 Medium-Risk Customers
Engagement campaigns
Personalized plan recommendations

🟢 Low-Risk Customers
Upselling premium services
Referral and loyalty programs
